## Search Track Demo Application
Application to search tracks and provide details for the track like `Artist Name`, `Number of listeners` and `Song Url`

### Application Architecture: 
MVP

### Laguage Used:
Java

### Api Used:
Search Track - Provived by [Last.fm](https://www.last.fm/api/show/track.search)

### Components Used:
- [x] AndroidX
- [x] RxJava2 - Networking
- [x] Retrofit - Networking
- [x] Dagger - DI
- [x] GSON - Json to Pojo
- [x] ButterKnife - DI
- [x] Glide - Image loader
- [x] Espresso - UI test cases

Application covers for UI test and Unit Tests for business logic.
